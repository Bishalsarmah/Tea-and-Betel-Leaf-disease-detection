# 🍃 AgriSight AI: Leaf Disease Diagnosis Dashboard

An automated, data-driven plant pathology dashboard built to assist small-scale farmers in identifying tea and betel leaf diseases. Utilizing an optimized **MobileNetV3-Large** backbone modified with **Efficient Local Attention (ELA_S)**, the system achieves **98.51% diagnostic accuracy** while maintaining a lightweight footprint suitable for real-time mobile deployability.

---

## 📊 Core Features
* **Dual-Crop Mapping:** Trained specifically to classify 10 conditions across Tea and Betel crop varieties.
* **Explainable AI (XAI):** Implements **Grad-CAM Saliency Maps** to show exactly where the neural filters are directing visual attention on the specimen matrix.
* **Confidence Metrics & Analytics:** Real-time probability vector distribution tracking with actionable treatment recommendations.
